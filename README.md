# GRIX

**Simple, private peer-to-peer audio & video communication for iOS.**

[![iOS](https://img.shields.io/badge/platform-iOS-blue.svg)](https://developer.apple.com/ios/)
[![Swift](https://img.shields.io/badge/language-Swift%205.9-orange.svg)](https://swift.org)
[![License](https://img.shields.io/badge/license-MIT-lightgrey.svg)](LICENSE)
[![App Store](https://img.shields.io/badge/App%20Store-Coming%20Soon-4CA4F6?logo=apple)](https://apps.apple.com/app/idYOUR-APP-ID)

> **No servers. No accounts. No data collection.**  
> Just direct, encrypted audio/video between two iPhones or iPads.

---

## Features

- **100% Peer-to-Peer** – Uses Apple’s `MultipeerConnectivity` framework
- **Zero Data Collection** – Nothing is stored or sent to any server
- **Local Network + Bluetooth** – Works over Wi-Fi or direct Bluetooth
- **No Sign-Up Required** – Open the app and connect

---

## Requirements

- iOS 26.0+
- MacOS 26.0+
